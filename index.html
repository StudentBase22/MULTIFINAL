What is Multimedia?

The integration of multiple forms of media into a single application.

�

Elements of Multimedia

  • Text
  • Graphics (Images)
  • Audio (Sound)
  • Video
  • Animation
⚡

Characteristics of Multimedia

  • Interactive
  • Digital
  • Integrates different media types
  • Enhances communication and learning
🔗

Hypertext vs Hypermedia

Hypertext Hypermedia
Text containing links to other text Extension of hypertext
Example: Web pages Includes text, images, audio, video, and animations connected through links
🌐

HTTP (HyperText Transfer Protocol)

Protocol used for communication between web browsers and web servers.

  • Foundation of data communication on the World Wide Web
  • Functions:
  • Request web pages
  • Transfer multimedia content
  • Enable client-server communication
📜

History of Multimedia

  • Early Stage: Text-only systems, limited graphics
  • Development Stage: Improved computers, better storage, digital audio and video
  • Modern Multimedia: Streaming media, VR and AR, interactive web applications

Sound and Audio

Sound characteristics, digital audio, MIDI, and audio file formats.

🔊

Sound

Energy produced by vibrations.

Sound Process: Vibration → Sound Wave → Ear → Brain

Characteristics:

  • Amplitude: Determines loudness
  • Frequency: Determines pitch
  • Wavelength: Distance between waves
🎧

Digital Audio

Sampling: Converting analog sound into digital form.

Sampling Rate: Number of samples taken per second. Common rate: 44.1 kHz

Bit Depth: Determines audio quality.

  • Higher bit depth: Better quality
  • Higher bit depth: Larger file size
🎹

MIDI

Musical Instrument Digital Interface

Purpose: Communication standard for electronic musical devices.

Advantages:

  • Small file size
  • Easy editing

Disadvantages:

  • Depends on playback device
  • Less realistic than recorded audio
📁

Audio File Formats

Format Characteristics
WAV High quality, Large size
MP3 Compressed, Small size
AAC Better compression, Common in streaming
MIDI Instructions rather than actual sound
